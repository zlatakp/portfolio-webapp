## 1. Editorial Contact Sheet
Implemented in this workspace.

An art-direction-led layout that feels like a photographer's proof deck: paper tones, dark ink text, warm accent color, cropped image windows, and package cards that read like curated print labels. Best when the brand needs to feel premium without becoming cold.

Key traits:
- Layered paper-and-film visual system
- Strong featured tier with ribbon treatment
- Package details framed as shoot deliverables, not generic pricing bullets

## 2. Gallery Mosaic
A more immersive direction where each package is paired with a different image crop and presented inside an offset gallery grid. This works well for a photographer brand that sells style and atmosphere first, with pricing revealed as part of the visual story.

Key traits:
- Asymmetric image mosaic behind tier cards
- Hover-led emphasis on one package at a time
- Strong lifestyle feel, weaker for fast comparison

## 3. Quiet Luxury Timeline
A restrained, high-end layout that compares packages as a shoot journey from planning to delivery. Packages are shown as three lanes across a horizontal timeline, emphasizing service depth over discount language.

Key traits:
- Minimalist composition with elegant typography
- Strong for wedding and portrait photographers
- Very clear service progression, less visually bold than the editorial route

## 4. Darkroom Noir
A cinematic direction that leans into black backgrounds, warm metallic accents, and spotlighted package cards. This is the boldest option and works for a brand that wants to feel fashion-led, moody, and premium.

Key traits:
- High-contrast art direction
- Dramatic featured-tier emphasis
- Strong atmosphere, still readable on mobile

## 5. Sunlit Storyboard
A bright, optimistic route that arranges packages like pinned notes from a shoot day: tape details, layered cards, and soft sunset gradients. It feels approachable and high-touch without becoming casual.

Key traits:
- Warm daylight palette
- Handcrafted editorial details
- Service information broken into easy, scannable blocks
